HTML

HTML — стандартизированный язык гипертекстовой разметки документов для просмотра веб-страниц в браузере.
Структура HTML
Основаня структурная единица в HTML - тег. Среди тегов выделяются две разновидности: закрывающиеся и одиночные.

Тег в HTML выглядит следующим образом: <название тега>.
А его закрывающий товарищ: </название тега>.

Самый главный тег для любого HTML документа - тег <html>.
В нём находится всё остальное содержимое документа. Включая метаданные и саму разметку.
Тег <html> является закрывающимся, поэтому имеет закрывающий тег </html>.

Примеры тегов в HTML
Для примера разберём несколько видов тегов и обсудим их предназначение.
  • Тег <p> является закрывающимся тегом абзаца или же параграфа. Он может включать в себя текст и другие элементы, например ссылки.

  • Тег <a> или же ссылка, также является закрывающимся тегом. Может содержать ссылки на другие части сайта или его страницы, а также ссылаться на другие сайты в Интернете.

  • Тег <img> является одиночным тегом. Он содержать изображения, встраиваемые на страницу.

  • Тег <div> яляется закрывающимся тегом. Он является одной из главных структурных единиц объединяя другие элементы и не имея собственной спецификации.

Атрибуты в HTML
Помимо тегов в HTML есть атрибуты. Они указываются внутри тега:
  • <link rel="stylesheet" href="пример.css">

  • <html lang="en">

  • <img src="img.png" alt="текст, если изображения нет">

"Надстройки" HTML
Чистый HTML не удовлетворяет требованиям современного пользователя, ибо скуден в визуальном оформлении и функционале.

Для решения проблемы визуального оформления основным средством стал CSS.
Таблица Каскадных Стилей, именно так расшифровывается и переводится абревиатура на русский, имеет возможность модифицировать разметку и оформление элементов.
CSS код можно писать прямо в HTML документе при помощи парного тега <style>, однако это считается мувитоном.
Правильным решением является хранение CSS кода в отдельном файле с расширением .css, подключённом к HTML документу тегом <link> с атрибутами rel="stylesheet" href="пример.css".

Для решения проблем с ограниченным функционалом используются разлчные языки программирвоания, например: JavaScript и Python.
